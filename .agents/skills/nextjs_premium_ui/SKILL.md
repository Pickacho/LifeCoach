# Premium Next.js UI Rules

## Aesthetics
1. **Glassmorphism**: Use semi-transparent backgrounds with subtle borders and heavy background blurs for cards and modals.
2. **Color Palette**: Stick to deep dark modes (e.g., `#0f172a`) with vibrant, glowing accents (e.g., `#3b82f6`, `#22d3ee`). Do not use raw, standard colors.
3. **Typography**: Ensure standard systemic fonts (`Inter`, `system-ui`) with varying weights. Emphasize tracking and line-height for readability.

## Components
1. **Micro-animations**: Elements should have subtle translation (e.g., `translateY(-2px)`) and box-shadow changes on `:hover`.
2. **No Placeholders**: Always present a functional or functional-looking demo rather than a generic `TODO` block.
