# Web Development Standards

## Architecture & Semantics
1. **Semantic HTML**: Use proper HTML5 semantic tags (`<header>`, `<nav>`, `<article>`, `<section>`, etc.) for accessibility and structure.
2. **SEO Optimization**: Ensure unique Title, Meta descriptions, Open Graph tags, and structured data (JSON-LD) for indexability.
3. **Core Web Vitals**: Optimize LCP, CLS, and INP metrics by deferring non-critical CSS/JS, lazy-loading images, and using modern formats (WebP/AVIF).

## Mobile-First & UI
1. **Responsive Design**: Build mobile-first. Use CSS Grid and Flexbox for fluid layouts.
2. **Aesthetics & Micro-interactions**: Implement polished hover states, focus indicators, and subtle transitions for a premium user experience.
